# i3 Dynamic Workspace Management

## Goal
Scripts/tools for dynamically creating and switching i3 workspaces, motivated by multi-worktree workflows.

## Steps

- [x] Bash script `scripts/i3-workspace` — switch to or create workspace by name, fzf picker when no arg
- [x] Symlink to `~/.local/bin/i3-workspace`
- [ ] Integrate with ulauncher (extension or shortcut)
- [ ] Workspace naming conventions (e.g. project/branch patterns)
- [ ] Auto-assign workspaces to outputs/monitors
